## Settings Panel

Press the gear icon to access the settings.

**Important: Only an administrator can set the TML serial numbers and locate the TML probes to box or windrow. Contact for changes Triologic or Compost Systems.**

### Batch

:::tip
If no batch is created at box/windrow, the temperature recording is inactive.
:::

<div class="imgSized0">
  ![Batched Setting 1](/img/operator_panel/batch_create.png)
</div>
<div class="imgSized0">
  ![Batched Setting 1](/img/operator_panel/batch_remove.png)
</div>

1. Enter batch name (can be free chosen)
2. Click Create Batch to set batch name and start temperature recording --> Temp. Record active
3. Click Take out Batch to stop temperature recording --> Temp. Record inactive

### Chart

<div class="imgSized0">![Batched Setting 1](/img/operator_panel/chart.png)</div>

1. Klick Show Chart in Full Screen to display chart in a new window.
2. Set the sanitization temperature which is shown as a red line in chart diagram.
3. Klick for export chart

<div class="imgSized3">

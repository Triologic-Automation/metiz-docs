## Description

The widget displays all measured values and status information of Windrow/Box and allows to change aeration and irrigation (optional) operation mode.

<ol class="liGreenNumbers">
	<li>Enter settings menu  </li>
	<li>Actual batch name  </li>
	<li>Measured temperature of sensor T3  </li>
	<li>Measured temperature of sensor T2  </li>
	<li>Measured temperature of sensor T1  </li>
	<li>Signal strength of the TML connection  </li>
	<li>Timestamp of the last data transmission  </li>
	<li>Pressure at aeration blower (OPTIONAL)  </li>
	<li>Door status (OPTIONAL)  </li>
	<li>Irrigation amount (OPTIONAL)  </li>
	<li>Time of next Irrigation cycle (OPTIONAL)  </li>
	<li>Status of the temperature recording  </li>
	<li>Number of windrow/box  </li>
</ol>

Aeration:  

<div class="A"> 
	<ol class="">
		<li>Fan symbol (rotates during operation)</li>
		<li>Manuel Start button (continuous operation)</li>
		<li>Stop button</li>
		<li>Automatic/temperature mode according to temperatures in material </li>
		<li>Interval mode</li>
	</ol>
</div>

Irrigation (OPTIONAL):  

<div class="B"> 
	<ol class="">
		<li>Status symbol of irrigation   </li>
		<li>Manuel Start button (continuous operation)  </li>
		<li>Stop button  </li>
		<li>Automatic mode  </li>
	</ol>
</div>

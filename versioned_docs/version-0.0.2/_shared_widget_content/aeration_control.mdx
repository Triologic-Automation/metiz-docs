### Aeration Control Details

- **Symbol [A0]**: The symbol changes the color depending on the status.

  - grey: no operation
  - green: operation
  - red: malfunction at blower

- **Manual start button (continuous operation) [A1]**: Manual start button for blower/fan. By pressing this button the blower/fan starts to operate. He runs until he gets stopped by the user by selecting another mode or by a malfunction.
- **Stop button [A2]**: By klick on this button the blower/fan gets stopped until another mode is chosen.
- **Interval Mode [A3]**: The blower/fan is working in interval mode. He gets switched on or off for a chosen time span. For settings of this mode, see [SettingsðŸ”—](#interval-mode).
- **Temperature/Automatic mode [A4]**: The blower/fan is working in temperature/automatic mode. This mode operates similarly to the interval mode but controlled by the measured temperatures (middle value of the 3 measures temperatures) in the material. For settings of this mode, see points [SettingsðŸ”—](#temperatureautomatic-mode-runtime-settings) and [SettingsðŸ”—](#temperatureautomatic-mode-pausetime-settings).

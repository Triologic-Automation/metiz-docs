### Batch

:::tip
If no batch is created at box/windrow, the temperature recording is inactive.
:::

<div class="imgSized0">

1. Enter batch name (can be free chosen)
2. Click Create Batch to set batch name and start temperature recording --> Temp. Record active
3. Click Take out Batch to stop temperature recording --> Temp. Record inactive

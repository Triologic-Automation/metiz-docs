The widget displays all measured values as well as status information and allows to change aeration operation mode.
This widget is basically a derivation from [**Windrow/Box Widget**](./windrow). This is why the same information applies.

<div class="imgSized1">![Blower Widget](/img/widgets_numbered/blower.png)</div>

<ol class="liGreenNumbers">
  <li>Enter settings menu </li>
  <li>Actual batch name </li>
  <li>Number of windrow/box </li>

  <li>Measured temperature of sensor T1 (Further info: [ðŸ”—TML](../metiz_general/tml.mdx)) </li>
  <li>Signal strength of the TML connection </li>

  <li>Status of the temperature recording </li>
</ol>

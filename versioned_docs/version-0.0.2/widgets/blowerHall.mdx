## Fan Control

<div class="A2">
  <ol class="">
    <li>Manuel Start button (continuous operation)</li>
    <li>Stop button</li>
    <li>Automatic mode</li>
  </ol>
</div>

- **Manual start button (continuous operation) [A1]**: Manual start button for blower. By pressing this button, the blower starts to operate. He runs until he gets stopped by the user by selecting another mode or by a malfunction.  
- **Stop [A2]**: By clicking on this button the blower gets stopped until another mode is chosen. 
- **Automatic mode [A3]**: The blower is working in automatic mode. The automatic program runs at a predefined speed during the day and at night. For settings of this mode, see _.

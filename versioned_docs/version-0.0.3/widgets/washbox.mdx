## Pump Control

<div class="A">
  <ol class="">
    <li>Manuel Start button (continuous operation)</li>
    <li>Stop button</li>
    <li>Automatic mode</li>
  </ol>
</div>

- **Manual start button (continuous operation) [A1]**: Manual start button for percolate pump. By pressing this button, the pump starts to operate. It runs until he gets stopped by the user by selecting another mode or by a malfunction. 
- **Stop [A2]**: By click on this button the pump gets stopped until another mode is chosen. 
- **Automatic mode [A3]**: If at one of the boxes/windrows is a Batch created and automatic mode is active, the percolate pump starts to work.
